# MCP Servers Monorepo

## 目的
Model Context Protocol (MCP) サーバーのコレクションを提供する。

## 要求
- 各MCPサーバーは独立してリリース可能であること
- 共通のビルド・テスト基盤を持つこと
- describe/execute パターンで操作を提供すること

## 対象ツール
1. mcp-git-repo-explorer - Gitリポジトリ操作
2. mcp-interactive-instruction - ドキュメント管理
3. mcp-traceable-chain - 依存関係付きドキュメントチェーン
